FitBuddy – это веб-приложение для поиска людей для совместных тренировок, участия в спортивных событиях и общения в чатах. Проект реализован в рамках учебной работы (PERN + Docker)

Живая версия доступна по адресу:  
[https://fitbuddy-3uoq.onrender.com]

Стек технологий

- Frontend: React (Create React App), React Router, CSS Modules / CSS
- Backend: Node.js, Express
- Database: PostgreSQL (с расширениями `cube`, `earthdistance` для геопоиска)
- Контейнеризация: Docker, Docker Compose
- Деплой: Render (Web Service + PostgreSQL Managed Database)
